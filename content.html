Task

Write a function named character_count with two arguments:

  • string: a string containing some characters.
  • character: a single character to count in the string.

Your function should return how many times character appears in string and return that value. Use the accumulator pattern to keep track of the count.

# Write your function here # These calls should print 2, 3 and 0 respectively print(character_count("A bunch of characters", "a")) print(character_count("To be or not to be, that is the question", "e")) print(character_count("Hello, World!", "4"))
